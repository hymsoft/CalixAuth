# Política de Privacidad de CalixAuth

**Última actualización:** 03 de Febrero de 2026

**HyM Soft** ("nosotros", "nuestro") ha desarrollado **CalixAuth** como una aplicación de código abierto y comercial. Este SERVICIO se proporciona tal cual es.

Esta página se utiliza para informar a los visitantes sobre nuestras políticas con la recopilación, el uso y la divulgación de información personal si alguien decide utilizar nuestro Servicio.

## 1. Recopilación y Uso de Información

**CalixAuth no recopila, transmite ni almacena ningún dato personal identificable.**

Nuestra filosofía es **"Usar y Olvidar"**. Toda la generación de claves y el procesamiento de datos ocurren localmente en su dispositivo.

### Datos del Dispositivo
La aplicación no utiliza servicios de terceros que puedan recopilar información utilizada para identificarlo.

### Permisos Solicitados
- **Portapapeles (Clipboard):** Requerido únicamente para copiar la contraseña generada. La aplicación limpia automáticamente el portapapeles después de 60 segundos (o cuando la aplicación pasa a segundo plano) por motivos de seguridad.
- **Hápticos (Vibración):** Se utiliza para proporcionar retroalimentación física al generar claves o interactuar con la interfaz.

## 2. Almacenamiento de Datos

CalixAuth está diseñada para **NO tener persistencia de datos sensibles**.
- Las contraseñas generadas se almacenan únicamente en la memoria volátil (RAM) del dispositivo.
- El historial de contraseñas se elimina automáticamente cuando la aplicación se cierra, pasa a segundo plano o se bloquea la pantalla.
- La configuración de preferencias (tema, idioma, longitud por defecto) se almacena localmente en el dispositivo utilizando `AsyncStorage` y no contiene información personal.

## 3. Seguridad

Valoramos su confianza. Utilizamos estándares de criptografía fuertes (`expo-crypto` que implementa CSPRNG) para la generación de claves. Sin embargo, recuerde que ningún método de transmisión por Internet o método de almacenamiento electrónico es 100% seguro y confiable, aunque nuestra aplicación funciona completamente offline para minimizar riesgos.

## 4. Enlaces a Otros Sitios

Este Servicio puede contener enlaces a otros sitios (por ejemplo, para soporte o calificación). Si hace clic en un enlace de un tercero, será dirigido a ese sitio. Tenga en cuenta que estos sitios externos no son operados por nosotros. Por lo tanto, le recomendamos encarecidamente que revise la Política de Privacidad de estos sitios web. No tenemos control ni asumimos ninguna responsabilidad por el contenido, las políticas de privacidad o las prácticas de sitios o servicios de terceros.

## 5. Privacidad de los Niños

Nuestros Servicios no se dirigen a menores de 13 años. No recopilamos a sabiendas información de identificación personal de niños menores de 13 años.

## 6. Cambios a esta Política de Privacidad

Podemos actualizar nuestra Política de Privacidad de vez en cuando. Por lo tanto, se le recomienda revisar esta página periódicamente para ver si hay cambios. Le notificaremos cualquier cambio publicando la nueva Política de Privacidad en esta página.
